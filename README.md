# Food-Munch-Website
Developed a fully responsive food ordering website using HTML, CSS, and Bootstrap, implementing mobile-first design principles and reusable UI components. The application features structured navigation, interactive sections, and optimized layout for cross-device compatibility.

---

📌 Overview

Food Munch is a fully responsive food ordering website built using HTML, CSS, and Bootstrap.
The project focuses on mobile-first design, structured UI components, and seamless user experience across different screen sizes.

---

🚀 Features

- 📱 Fully responsive design (mobile, tablet, desktop)
- 🧭 Structured navigation with smooth scrolling
- 🍽️ Interactive menu sections with categorized items
- 🎯 Clean and reusable UI components using Bootstrap
- 💳 Delivery & payment section with visual elements
- 🎁 Modal-based user interaction (gift/redeem feature)
- 🌐 Cross-browser compatibility

---

🛠️ Tech Stack

- HTML5 – Semantic structure
- CSS3 – Styling and layout customization
- Bootstrap 4 – Responsive grid & components
- Font Awesome – Icons
- JavaScript (Bootstrap JS) – Interactivity
